## 指针的定义以及初始化  
```C
int * pi;
char * pc;
float * pf, * pg;
int * pi=指针;
int * p = NULL;// 最好给它赋初值,否则它就会乱指
```
*和指针名之前的空格可有可无.通常在声明之前使用空格,在解引用变量时省略空格.  
大部分系统内,指针由一个无符号整数表示,但它并不是整数类型.  
**指针得先直向一个变量才能使用,必须先初始化才能给`*p`赋值**
